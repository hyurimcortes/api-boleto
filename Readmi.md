
# Projeto Validar boleto bancario

Este projeto tem como objetivo validar codigo de barra de boleto bancario tanto de concecionaria quanto bancario.


Projeto simples, estrutural e funcional.


# Como iniciar o projeto

 - Clonar
 - Yarn install ou npm install
 - Yarn dev ou npm run dev
